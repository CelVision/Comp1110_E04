# README
How to Modify nodes.txt
Keep the original building list in nodes.txt.
For each building, add comma-separated neighboring buildings after its name (refer to Google Maps for walking adjacency).
Example (only as example):
Centennial Gate, Central Podium Levels(cpd), Cheng Yu Tung Tower
Commit changes to GitHub after checking no extra spaces/format errors.

Purpose of nods.txt
This file stores building names and their adjacent nodes. The program will read these data and automatically generate new line segment classes for campus path visualization and logic processing
